## Előkészületek

### Töltsd le a projektet

```sh
git clone https://github.com/refilc/teacher
cd naplo
```

### Telepítsd a könyvtárakat

```sh
bash fix-pub.sh
```

### Futtasd az applikációt

```sh
cd filcnaplo
flutter run
```
